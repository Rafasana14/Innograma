¡Bienvenidos a Innograma!

Innograma es una aplicación destinada para ser utilizada por parte del comité de programa de Innosoft. El objetivo principal de Innograma es permitir llevar a cabo la gestión de ponencias, ponentes y eventos que se llevarán a cabo durante los Innosoft Days de cada curso académico
